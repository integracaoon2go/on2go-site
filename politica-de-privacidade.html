Política de Privacidade

A On2Go leva a sua privacidade a sério. Esta política descreve como tratamos os dados através das nossas automações no Instagram.

1. Coleta de Dados

O nosso aplicativo processa mensagens diretas (DMs) e comentários enviados por você para fornecer respostas automáticas inteligentes via API oficial da Meta.

2. Uso das Informações

Os dados (como ID do usuário e texto da mensagem) são utilizados exclusivamente para gerar a resposta do robô e melhorar o atendimento. Não compartilhamos esses dados com terceiros para fins publicitários.

3. Segurança

Todas as comunicações são feitas através de canais seguros e criptografados fornecidos pela infraestrutura da Meta.

4. Contato

Para dúvidas sobre seus dados, entre em contato através do nosso Instagram oficial.
